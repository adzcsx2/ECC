# /ecc:crosspost

跨平台内容分发，将同一内容适配到 X、LinkedIn、Threads 和 Bluesky 等不同平台。

---

## 功能

- 按平台约束自动适配内容，保持作者声音一致
- 内置反模式过滤：禁止「Excited to share」等模板化表达
- 整合 brand-voice 和 content-engine 进行声音捕捉和内容优化
- 输出包含各平台版本、变更说明和未解决的发布约束

## 用法

- `/ecc:crosspost` - 将同一个想法分发到多个社交平台时，生成各自适配的版本

## 适用场景

- 产品发布、更新或文章需要在多个平台发布
- 需要保持作者声音一致且各平台内容不重复
- 用户提到"crosspost""post this everywhere"等关键词
