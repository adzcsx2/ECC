# /ecc:perl-patterns

现代 Perl 5.36+ 习语、Moo OO 模式及最佳实践。

---

## 功能

- `use v5.36` 统一前导与子程序签名
- Moo 轻量级 OO 框架与角色（Role）模式
- 后置解引用、命名捕获等现代 Perl 语法
- 文件 I/O 三参数 open、Path::Tiny 模式
- perltidy / perlcritic 工具链配置

## 用法

- `/ecc:perl-patterns` - 现代 Perl 开发模式参考

## 适用场景

- 编写新的 Perl 代码或模块
- 将遗留 Perl 代码重构为现代标准
- Perl 模块架构设计
