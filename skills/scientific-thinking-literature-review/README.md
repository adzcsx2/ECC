# /ecc:literature-review

系统文献综述：检索规划、来源筛选、综合分析与证据记录。

---

## 功能

- 四种综述类型：叙述性、范围性、系统性、Meta 分析
- PICO 框架与研究问题定义
- 多数据库检索（PubMed、arXiv、Semantic Scholar）
- 去重策略：DOI > PMID > 标题 > 标准化标题
- 结构化数据提取与置信度分层综合

## 用法

- `/ecc:literature-review` - 进行系统性学术文献综述

## 适用场景

- 撰写研究论文的背景综述
- 系统性或范围性文献调研
- 发现研究空白与矛盾证据
