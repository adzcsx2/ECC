# /ecc:motion-advanced

React/Next.js 高级动效模式：拖拽、手势、文字动画、SVG 路径绘制、自定义 hooks 和命令式动画序列。

---

## 功能

- 拖拽交互：可拖拽卡片、拖拽关闭面板、Reorder.Group 排序列表
- 手势检测：滑动检测、长按、取景框缩放
- 文字动画：逐词揭示、逐字打字、数字计数器
- SVG 动画：路径渐显、图标形变、圆形进度环
- 自定义 hooks：`useScrollReveal`、`useHoverScale`、光标跟随
- 命令式序列：通过 `useAnimate` 实现可中断的多步动画
- 加载器组件：骨架屏、脉冲点、进度条、按钮加载态

## 用法

- `/ecc:motion-advanced` - 构建需要拖拽、手势、文字动画或 SVG 绘制的复杂动效

## 适用场景

- 构建可拖拽排序列表、滑动关闭面板或磁吸按钮
- 实现逐词入场文字、逐字打字效果或数字跳动
- SVG 路径动画、图标形变、环形进度条等视觉交互
- 编写可复用的动画 hook（滚动揭示、悬停缩放、光标追踪）
- 需要多步命令式动画序列时（先决条件：已配置 `motion-foundations`）
