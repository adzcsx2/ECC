# /ecc:carrier-relationship-management

货运承运商关系管理的专业知识库，涵盖承运商组合管理、费率谈判和绩效追踪。

---

## 功能

- 承运商全生命周期管理：寻源、合规审查（FMCSA SAFER）、合同谈判、路由指南构建
- 费率谈判框架：拆解线路运费、燃油附加费、附加费、最低收费等独立组件
- 承运商绩效评分卡：准时交付率、招标接受率、索赔率、发票准确率等核心指标
- RFP 全流程指导：从投标包设计到评估权重分配再到分波次授标与实施
- 市场情报：DAT/Greenscreens 费率基准、货运市场周期分析、季节性模式

## 用法

- `/ecc:carrier-relationship-management` - 获取承运商管理的策略建议、决策框架和操作指南

## 适用场景

- 管理承运商组合、谈判运费费率时
- 运行年度或按线路的 RFP 进行费率基准测试时
- 承运商绩效评估与业务量重新分配
- 运力紧张期间的应急承运商激活与费率谈判
