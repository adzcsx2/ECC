# /ecc:cpp-coding-standards

基于 C++ Core Guidelines 的现代 C++ 编码规范，确保类型安全、资源安全和代码清晰。

---

## 功能

- 覆盖哲学、函数、类、资源管理、错误处理等全面的 C++ 规范
- 强制 RAII、不可变性、类型安全和 constexpr 等现代 C++ 实践
- 提供 DO/DON'T 对比示例，避免常见反模式
- 包含快速检查清单，审核代码前逐项核对

## 用法

- `/ecc:cpp-coding-standards` - 编写、审查或重构 C++ 代码时获得规范指导

## 适用场景

- 编写新的 C++ 类、函数或模板
- 审查或重构现有 C++ 代码
- 在 C++ 项目中做架构决策或选择语言特性
