# /ecc:laravel-patterns

生产级 Laravel 架构模式：路由、Eloquent ORM、服务层、队列、事件、缓存和 API 资源。

---

## 功能

- 薄控制器 + 服务层 + Action 类的分层架构
- 路由模型绑定和范围绑定（scope bindings）
- Eloquent 模型最佳实践（casts、value objects、scope、eager loading）
- Query 对象模式实现复杂筛选
- 数据库迁移、表单请求验证、API 资源、事务管理
- 队列、事件、缓存和配置管理

## 用法

- `/ecc:laravel-patterns` - 构建 Laravel 应用、设计 API、优化 Eloquent 查询

## 适用场景

- 构建 Laravel Web 应用或 API
- 规范控制器、服务和领域逻辑结构
- 使用 Eloquent 模型和关系
- 设计 API 资源和分页
- 添加队列、事件、缓存和后台任务
