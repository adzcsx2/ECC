# /ecc:enterprise-agent-ops

长期运行 agent 的生命周期管理、可观测性、安全边界与变更管理。

---

## 功能

- 运行时生命周期控制：启动、暂停、停止、重启 agent 工作负载
- 可观测性：日志、指标、追踪的基线采集与监控
- 安全控制：最小权限凭证、环境级密钥注入、硬超时与重试预算、高风险操作审计日志
- 变更管理：不可变部署工件、灰度发布、回滚与审计追踪
- 故障响应模式：冻结新发布、捕获代表性 trace、隔离故障路径、最小化修复、回归验证

## 用法

- `/ecc:enterprise-agent-ops` - 在管理云托管或持续运行的 agent 系统，需要运维控制时激活

## 适用场景

- 部署和管理长期运行的 AI agent 服务
- 建立 agent 系统的可观测性与监控体系
- 配置 agent 的安全权限边界和审计日志
- 制定 agent 系统的变更发布与回滚策略
- 排查 agent 运行中的批量故障
