# /ecc:hermes-imports

将本地 Hermes 操作流程转换为可公开发布的 ECC 技能与发布包。

---

## 功能

- 将本地路径转换为仓库相对路径或占位符
- 替换真实账户名为角色标签，避免泄露私密信息
- 移除凭据、令牌、OAuth 文件、个人联系方式等敏感数据
- 提供系统化的脱敏检查清单（绝对路径、密钥、电话号码、财务数据等）
- 输出标准化的转换合约：技能名称、脱敏摘要、所需公开输入、已移除私密输入、残留风险

## 用法

- `/ecc:hermes-imports` - 将重复使用的 Hermes 工作流脱敏后导入 ECC，确保不泄露私密工作区状态

## 适用场景

- Hermes 工作流重复使用次数足够多，适合公开化
- 本地操作员提示词需转化为公开 ECC 技能
- 启动、内容、研究或工程工作流需要脱敏交接文档
- 工作流提及本地路径、凭据、私人账户名等必须移除后才能发布
