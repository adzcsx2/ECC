# /ecc:angular-developer

生成 Angular 代码并提供架构指导和最佳实践。

---

## 功能

- 创建或脚手架搭建 Angular 项目、组件、服务、指令、管道、守卫
- 实现响应式编程：Signals、linkedSignal、resource、effect
- 处理表单（信号表单、响应式表单、模板驱动表单）和表单验证
- 配置依赖注入、路由、懒加载
- 添加无障碍支持（ARIA）、动画、组件样式和 Tailwind CSS
- 编写 Angular 测试（单元测试、组件 harness、E2E）

## 用法

- `/ecc:angular-developer` - 创建 Angular 项目或生成符合最佳实践的组件和服务

## 适用场景

- 在任何 Angular 项目中工作
- 搭建新 Angular 项目或库
- 实现 Angular Signals 响应式编程
- 配置路由、依赖注入、SSR
- 编写或调试 Angular 测试
