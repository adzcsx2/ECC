# /ecc:brand-voice

从真实素材中构建写作风格画像，确保内容输出保持一致的品牌语调。

---

## 功能

- 从推文、文章、发布说明、产品文案等真实素材中提取风格特征
- 生成可复用的 VOICE PROFILE，供下游技能直接消费
- 提取节奏、句式、压缩度、标点习惯、语气尖锐度等关键特征
- 内置硬性禁用规则：自动拦截虚假好奇心、LinkedIn 八股、空洞形容词等
- 提供 ECC/Affaan 默认风格作为冷启动参考

## 用法

- `/ecc:brand-voice` - 收集指定作者/品牌的真实内容，生成结构化的语音风格画像

## 适用场景

- 需要跨渠道保持一致的写作风格时
- 为 X、LinkedIn、邮件、发布帖等多平台内容提供统一的语调基准
- 在 content-engine、crosspost、lead-intelligence 等技能之前调用，确保下游输出风格统一
