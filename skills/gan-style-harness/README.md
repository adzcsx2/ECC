# /ecc:gan-style-harness

基于 GAN 对抗网络的 Generator-Evaluator 多智能体架构，自主构建高质量应用。

---

## 功能

- 分离生成器与评估器角色，形成对抗性反馈循环，质量远超单智能体
- 三层代理架构（规划器 + 生成器 + 评估器），从一句话提示构建完整应用
- 评估器通过 Playwright 测试真实运行的应用，而非代码本身
- 四项评分标准（设计质量、原创性、工艺、功能性），可配置阈值和迭代次数
- 支持全栈应用构建和纯前端设计两种模式

## 用法

- `/ecc:gan-style-harness` - 使用三层代理架构从提示词构建高质量应用。可通过环境变量配置最大迭代次数、通过阈值、评估标准等

## 适用场景

- 从一个简短提示词构建完整可工作的应用
- 追求生产级质量的前端设计任务
- 需要经过验证的功能而非仅仅生成代码的全栈项目
- 对"AI 模板感"审美不可接受的项目
- 愿意投入 $50-200 换取生产级输出的任务
