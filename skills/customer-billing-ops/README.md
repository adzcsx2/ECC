# /ecc:customer-billing-ops

处理客户计费运营工作流，包括订阅管理、退款、流失处理和计费门户恢复。

---

## 功能

- 客户身份识别：从邮箱、Stripe ID、订阅号等标识符快速定位
- 问题分类体系：区分重复订阅、团队购买、支付失败、产品故障等场景
- 安全操作优先级：先恢复自助管理、再修复计费状态、最后退款
- 结构化输出：客户状态摘要、决策依据、行动记录、后续跟进和产品差距

## 用法

- `/ecc:customer-billing-ops` - 处理客户计费问题时，获得结构化的排查和操作流程

## 适用场景

- 客户反馈计费异常、要求退款或无法取消订阅
- 调查重复订阅、意外扣费、续费失败或流失风险
- 审计涉及订阅、发票、退款或支付方式的客服投诉
