# /ecc:ui-demo

使用 Playwright 录制精美 UI 演示视频，带光标和字幕。

---

## 功能

- 三阶段流程：Discover（探索 UI） -> Rehearse（彩排选择器） -> Record（录制）
- 注入 SVG 箭头光标覆盖层，支持平滑鼠标移动
- 字幕栏：Step N - Action 格式，渐进式展示
- 自然节奏：登录后 4s 停顿，按钮点击后 2s，逐字输入 25-40ms
- 可视化输入、平滑滚动、仪表板元素平移

## 用法

- `/ecc:ui-demo` - 录制 Web 应用演示视频

## 适用场景

- 制作产品功能演示视频
- 文档/教程的屏幕录制
- 利益相关者展示的走查视频
