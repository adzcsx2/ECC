# /ecc:production-scheduling

离散和批量制造的生产调度体系化专业知识，涵盖瓶颈管理、换线优化、中断响应和排程策略。

---

## 功能

- 瓶颈识别和鼓-缓冲-绳（DBR）约束管理
- 换线优化：SMED 方法论、作业顺序矩阵和 campaign vs mixed-model 决策
- 中断响应框架：设备故障、物料短缺、质量搁置、缺勤的重新排程
- OEE 整体设备效率分析和性能指标追踪
- ERP/MES 交互模式和计划与实际的偏差管理
- 劳动力管理：班次模式、技能矩阵和工会规则约束

## 用法

- `/ecc:production-scheduling` - 当生产订单竞争受限工序、中断需要快速重新排程、或需要在现有排程中插入新订单时使用

## 适用场景

- 生产订单争夺受限工作中心
- 设备故障或人员缺勤需要快速重新排程
- 换线和批量生产的经济决策
- 在现有排程中插入紧急订单
- 瓶颈发生变化需要重新分配瓶颈工序
- 制定生产计划和作业优先级排序
