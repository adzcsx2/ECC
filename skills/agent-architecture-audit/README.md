# /ecc:agent-architecture-audit

对 AI agent 应用进行 12 层全栈诊断审计。

---

## 功能

- 审计 agent 系统的 12 层架构：系统提示、会话历史、长期记忆、蒸馏、主动召回、工具选择、工具执行、工具解释、答案塑形、平台渲染、隐藏修复循环、持久化
- 检测包装器退化 —— 裸模型正确但包装后的 agent 出错
- 发现记忆污染、工具纪律失效和渲染损坏
- 生成按严重程度排序的诊断报告并提供代码级修复方案

## 用法

- `/ecc:agent-architecture-audit` - 对 agent 或 LLM 应用运行全栈诊断审计

## 适用场景

- 将 agent 或 LLM 应用发布到生产环境前
- Agent 行为在添加包装层后退化
- 同一模型在 playground 正常但在包装器中出错
- Agent 调试超过 15 分钟仍未找到根因
- 怀疑存在隐藏的修复/重试循环
