# /ecc:manim-video

使用 Manim 制作技术解说动画：流程图、系统架构、网络图谱和产品演示。

---

## 功能

- 结构化工作流（核心视觉论点 -> 分镜 -> 场景规划 -> 渲染 -> 打磨）
- 场景规划规则（每场景证明一个点、渐进式揭示、动效表达状态变化）
- 网络图谱默认模板（assets/network_graph_scene.py 可复用起点）
- 16:9 横向输出，低质量烟雾测试 -> 高质量最终渲染

## 用法

- `/ecc:manim-video` - 为技术概念生成精美的 Manim 动画解说视频

## 适用场景

- 需要技术解说动画
- 概念涉及图谱、工作流、架构或系统图
- 需要为 X 平台或落地页制作产品/发布解说
- 视觉效果需要精准而非普通电影感
