# /ecc:finance-billing-ops

收入、定价、退款与团队计费逻辑的代码级真相审计，区分销售事实与产品实现。

---

## 功能

- 从最新计费数据（Stripe 销售、订阅、退款、争议、重复订阅）入手建立全貌
- 区隔客户个案与产品真值：重复结账、团队意图、自助控制缺失、未兑现产品价值
- 检查代码路径验证计费行为：结账流程、定价页面、权益计算、席位/配额处理
- 输出结构化报告：销售快照、客户影响、产品真值、运营决策、产品缺口

## 用法

- `/ecc:finance-billing-ops` - 当需要查看收入快照、比较定价、诊断重复扣费或验证计费逻辑实现时激活

## 适用场景

- 查看 Stripe 销售、退款、MRR 或近期客户活动
- 验证团队计费、按席位计费在代码中是否真正生效
- 对比竞品定价或定价模式基准
- 排查计费相关客户投诉的根因
- 发现定价页面描述与实际代码行为的不一致
