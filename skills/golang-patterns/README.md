# /ecc:golang-patterns

Go 语言惯用模式与最佳实践，构建稳健高效的 Go 应用。

---

## 功能

- 核心设计原则：零值可用性、接受接口返回结构体、简洁优先于聪明
- 错误处理模式：错误包装添加上下文、自定义错误类型、errors.Is/As 检查
- 并发模式：工作池、Context 超时取消、优雅关闭、errgroup 协调、避免 goroutine 泄漏
- 接口设计：小接口组合、在使用方定义接口、类型断言可选行为
- 包组织与内存优化：标准项目布局、函数式选项模式、切片预分配、sync.Pool

## 用法

- `/ecc:golang-patterns` - 获取 Go 惯用模式指导，涵盖错误处理、并发、接口设计、包组织和性能优化

## 适用场景

- 编写新的 Go 代码
- 审查 Go 代码
- 重构现有 Go 代码
- 设计 Go 包/模块
