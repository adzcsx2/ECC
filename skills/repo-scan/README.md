# /ecc:repo-scan

跨技术栈源码资产审计：文件分类、第三方库检测与可交互 HTML 报告。

---

## 功能

- 跨栈扫描（C/C++、Java/Android、iOS、Web）一次完成
- 文件分类：项目代码、第三方代码、构建产物
- 50+ 已知库检测（FFmpeg、Boost、OpenSSL 等）含版本提取
- 四级裁决：核心资产 / 提取合并 / 重建 / 废弃
- 可交互暗色主题 HTML 报告

## 用法

- `/ecc:repo-scan` - 对代码仓库进行全量资产审计

## 适用场景

- 接手遗留大型代码库需要结构概览
- 重构前识别核心代码与死代码
- 审计嵌入源码中的第三方依赖
