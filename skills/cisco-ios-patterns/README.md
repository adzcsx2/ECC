# /ecc:cisco-ios-patterns

Cisco IOS/IOS-XE 配置审查、只读命令收集与安全变更验证模式。

---

## 功能

- IOS 配置模式参考：全局配置、接口、路由进程、线路配置模式
- 只读信息收集清单：show 命令集，只取所需段落而非导出完整配置
- 通配符掩码校验：防止子网掩码误用作通配符掩码导致匹配范围异常
- ACL 放置审查：方向确认、管理流量保护、回滚路径与计数器检查
- 变更窗口验证：变更前后的状态对比与行为校验

## 用法

- `/ecc:cisco-ios-patterns` - 获取 Cisco 设备配置审查和变更验证的操作模式

## 适用场景

- 审查计划变更前的 IOS/IOS-XE 配置
- 选择故障排查用只读 show 命令
- 检查 ACL 通配符掩码和接口方向
- 验证变更是否落地到 running-config 并确认保存
