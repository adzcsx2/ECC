# /ecc:ecc-tools-cost-audit

ECC Tools 成本审计：排查 PR 泛滥、配额绕过、付费模型泄漏与重复作业烧钱问题。

---

## 功能

- 追踪 webhook 入口到队列消费者的完整请求路径，定位烧钱根因
- 审计 PR 乘法效应：检查分支命名、去重逻辑与递归分析风险
- 排查配额绕过：对比配额检查点与用量扣减点的时序差异
- 检测付费模型泄漏：确认免费用户是否会路由到高价模型分析路径
- 分析重试烧钱：验证非瞬态错误的重复消费行为
- 按烧钱影响排序修复：先堵 PR 泛滥 > 配额绕过 > 模型泄漏 > 重复作业

## 用法

- `/ecc:ecc-tools-cost-audit` - 当怀疑 ECC Tools GitHub App 存在烧钱、超额创建 PR、用量绕过或 premium 模型泄漏时激活

## 适用场景

- ECC Tools 的账单异常增长
- 同一仓库被反复创建大量 PR
- 免费用户似乎在使用付费模型分析
- 用量限额被绕过导致超额扣费
- 分析作业失败但仍在消耗 token
