# /ecc:design-system

生成或审计设计系统，检查视觉一致性，审查涉及样式的 PR。

---

## 功能

- 模式一 - 生成设计系统：扫描代码库提取颜色/字体/间距/阴影，生成 DESIGN.md + 设计令牌 JSON + 可视化预览页
- 模式二 - 视觉审计：从颜色一致性、字体层级、间距节奏等 10 个维度评分，精确定位问题文件和行号
- 模式三 - AI 痕迹检测：识别 AI 生成设计中的通用模式（渐变滥用、蓝紫默认、无意义毛玻璃等）

## 用法

- `/ecc:design-system` - 新项目创建设计系统、审计现有 UI 一致性或检测 AI 痕迹

## 适用场景

- 启动新项目需要设计系统
- 审计现有代码库的视觉一致性
- UI 看起来"不对劲"但无法定位问题
- 审查涉样式的 PR
