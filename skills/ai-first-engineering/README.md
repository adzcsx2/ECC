# /ecc:ai-first-engineering

AI 辅助团队的工程运作模型：流程、审查和架构设计。

---

## 功能

- 定义 AI 优先团队的流程转变：规划质量优于编码速度、评估覆盖优于主观信心
- 设计 agent 友好的架构：显式边界、稳定契约、类型化接口、确定性测试
- 调整代码审查焦点：行为回归、安全假设、数据完整性、故障处理、发布安全
- 构建 AI 生成代码的高标准测试：强制回归覆盖、显式边界断言、接口集成检查
- 定义 AI 优先工程师的评估信号：分解模糊任务、定义可度量验收标准、产出高信号提示

## 用法

- `/ecc:ai-first-engineering` - 为 AI 辅助团队设计流程、审查标准和 agent 友好架构

## 适用场景

- 团队中 AI agent 产出了大量实现代码
- 设计使 agent 更高效的架构
- 为 AI 生成代码建立更严格的测试标准
- 招聘和评估 AI 优先工程师
