# /ecc:perl-testing

Perl 测试模式：Test2::V0、prove、Devel::Cover 与 TDD 工作流。

---

## 功能

- Test2::V0 现代测试框架：深度比较、构建器、子测试
- prove 运行器：并行测试、状态保存、CI 集成
- 模拟（Mocking）：Test::MockModule 与 Test::MockObject
- Devel::Cover 覆盖率：HTML 报告、CI 阈值检查
- TDD RED-GREEN-REFACTOR 完整工作流

## 用法

- `/ecc:perl-testing` - Perl 测试驱动开发与覆盖率指南

## 适用场景

- 为新 Perl 模块编写测试套件
- 从 Test::More 迁移到 Test2::V0
- 配置 prove + Devel::Cover 覆盖率流水线
