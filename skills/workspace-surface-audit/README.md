# /ecc:workspace-surface-audit

审计当前工作区：仓库、MCP、插件、环境变量，推荐 ECC 原生能力。

---

## 功能

- 四维盘点：仓库表面 -> 环境变量 -> 连接工具 -> ECC 现有能力
- 对标官方插件：分析 ECC 已有对等/仅原语/完全缺失
- 缺口推荐：按 Skill/Hook/Agent/MCP 正确形态拆分
- Top 3-5 下一步行动，按影响力排序
- 只读审计，不修改文件

## 用法

- `/ecc:workspace-surface-audit` - 审计工作区能力与推荐设置

## 适用场景

- 初次设置 Claude Code 时了解可用能力
- 对比官方插件与 ECC 覆盖度
- 决定新能力应以 Skill/Hook/Agent/MCP 哪种形态实现
