# /ecc:video-editing

AI 辅助视频编辑工作流——从原始素材到最终成片，各工具分层协作。

---

## 功能

- 六层编辑流水线：捕获 -> 组织 -> 精确剪切 -> 可编程合成 -> 生成素材 -> 最终润色
- 使用 Claude/Codex 分析转录文本、规划结构和生成编辑决策列表
- FFmpeg 负责精确剪切、批量裁剪、代理文件生成和音量标准化
- Remotion 处理文字叠加、数据可视化、动态图形等可编程合成
- ElevenLabs 和 fal.ai 用于声线、背景音乐、缩略图等 AI 生成素材

## 用法

- `/ecc:video-editing` - 剪辑视频素材、制作 vlog 或教程视频

## 适用场景

- 将长录制转换为短视频内容
- 从原始素材制作 vlog、教程或演示视频
- 为视频添加叠加字幕、背景音乐或声线
- 为不同平台（YouTube、TikTok、Instagram）重新构图
