# /ecc:video-editing

AI 辅助视频编辑：从原始素材到最终成片的六层流水线。

---

## 功能

- 六层流水线：Capture -> Organize -> FFmpeg 剪切 -> Remotion 合成 -> AI 资产生成 -> 人工调色
- FFmpeg 批量剪辑、拼接、代理生成与场景检测
- Remotion 可编程叠加层：文字、Logo、数据可视化
- ElevenLabs 配音 / fal.ai 音乐音效 / VideoDB 生成媒体
- 多平台尺寸重构图：YouTube 16:9、TikTok 9:16、Instagram 1:1

## 用法

- `/ecc:video-editing` - AI 辅助视频编辑工作流

## 适用场景

- 将长录制转为短视频内容
- 制作 Vlog、教程或演示视频
- 多平台（YouTube/TikTok/Instagram）发布适配
