# /ecc:mle-workflow

生产级机器学习工程工作流，涵盖数据契约、可复现训练、模型评估、部署、监控与回滚。

---

## 功能

- 将模糊的预测需求转化为显式的产品契约和数据契约
- 构建可复现的训练管线：种子、配置哈希、版本化制品
- 定义自动化晋升门槛（promotion gates）：离线指标、切片检查、延迟预算
- 完整运维闭环：特征/预测漂移监控、金丝雀发布、回滚路径
- 错误分析循环：将错误聚类为可实验的假设，而非笼统的"优化模型"

## 用法

- `/ecc:mle-workflow` - 构建、审查或加固生产 ML 系统，包括训练管线、评估、部署和监控

## 适用场景

- 规划排名、推荐、分类、预测、嵌入等生产 ML 项目
- 将 notebook 代码转化为可复现的训练/评估/推理管线
- 调试数据漂移、标签泄漏、特征陈旧或 serving 不一致问题
- 设计模型金丝雀发布、影子流量、自动回滚和运维看板
