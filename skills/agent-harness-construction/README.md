# /ecc:agent-harness-construction

设计并优化 AI agent 的动作空间、工具定义和观测格式。

---

## 功能

- 设计 agent 的动作空间：工具命名、输入 schema 和输出形状
- 优化观测设计：每个工具响应包含状态、摘要、后续操作和产出物
- 构建错误恢复契约：根因提示、安全重试指令和显式停止条件
- 管理上下文预算：保持系统提示简洁、按需加载技能、阶段边界压缩
- 提供架构模式指导：ReAct、函数调用、混合模式的选择与对比

## 用法

- `/ecc:agent-harness-construction` - 优化 agent 的规划、工具调用、错误恢复和收敛能力

## 适用场景

- 改进 agent 如何规划、调用工具、从错误恢复并收敛到完成
- 设计工具的粒度和边界（微工具 / 中工具 / 宏工具）
- 减少 agent 的 token 消耗和重试次数
- 跟踪完成率、pass@1/pass@3 和每次成功任务的成本
