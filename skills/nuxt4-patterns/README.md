# /ecc:nuxt4-patterns

Nuxt 4 应用的水合安全、路由规则与性能优化模式。

---

## 功能

- SSR 水合安全：避免服务端/客户端渲染不一致
- 数据获取：useFetch / useAsyncData 的最佳实践
- 路由规则配置：prerender、SWR、ISR、client-only
- 懒加载与性能优化：组件级 Lazy 加载与懒水合

## 用法

- `/ecc:nuxt4-patterns` - Nuxt 4 应用的架构与性能模式指南

## 适用场景

- 构建 Nuxt 4 SSR/SSG 应用
- 排查水合不匹配问题
- 配置路由级别的渲染策略
- 优化页面加载性能
