# /ecc:content-engine

从真实素材出发构建平台原生内容系统，拒绝空洞的 AI 套话。

---

## 功能

- 素材优先工作流：从文章、笔记、产品演示、文档中提取真实素材再创作
- 多平台适配：X、LinkedIn、短视频、YouTube、Newsletter 的平台特定写作规则
- 内容复用管线：从一个锚定资产中提取 3-7 个核心观点，逐个适配各平台
- 硬性禁用词库：自动拦截"在当今快速发展的..."、"颠覆性"等 AI 套路用语
- 品质门槛：每条内容必须有真实主张或证据点，严禁空洞填充

## 用法

- `/ecc:content-engine` - 基于真实素材，为多平台生成本地化的内容草稿和发布计划

## 适用场景

- 撰写 X 帖子或长线程
- 起草 LinkedIn 帖子或产品发布公告
- 将文章、播客、产品演示等内容复用为全平台内容
- 围绕产品或洞察构建发布序列和持续内容系统
