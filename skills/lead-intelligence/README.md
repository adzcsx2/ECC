# /ecc:lead-intelligence

AI 驱动的潜在客户发现和外联管道：信号评分、社交图谱分析和多渠道个性化外联。

---

## 功能

- 五阶段管道：信号评分 -> 互相排名 -> 暖路径发现 -> 资料丰富 -> 外联草稿
- 多重信号评分模型（职位、行业、活跃度、影响力、地理位置）
- X 和 LinkedIn 社交图谱分析，找到最佳引荐路径
- 多渠道外联（邮件、LinkedIn、X DM），适配目标渠道风格
- 内置反模式保护（无泛化模板、非真实熟悉感、非大量群发）

## 用法

- `/ecc:lead-intelligence` - 查找、评分并制定高价值联系人的个性化外联策略

## 适用场景

- 寻找特定行业的潜在客户或合作伙伴
- 建立销售、合作或融资的外联名单
- 分析社交图谱寻找暖引荐路径
- 为关键联系人撰写个性化外联文案
