# /ecc:market-research

市场研究、竞争分析、投资人尽调和行业情报，带来源归属和决策导向摘要。

---

## 功能

- 多种研究模式（投资人/基金尽调、竞争分析、市场估量、技术/供应商研究）
- 强制来源归属，区分事实、推断和建议
- 自下而上的市场估量校验（TAM/SAM/SOM）
- 包含反面证据和下行风险的平衡分析
- 标准化输出格式（执行摘要 -> 关键发现 -> 影响 -> 风险 -> 建议 -> 来源）

## 用法

- `/ecc:market-research` - 进行市场调研、竞争分析和投资尽调

## 适用场景

- 研究市场、品类、公司、投资人或技术趋势
- 构建 TAM/SAM/SOM 估算
- 比较竞争对手或相邻产品
- 在接触投资人前准备背景调查报告
- 进入市场前验证商业假设
