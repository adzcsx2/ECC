# /ecc:laravel-security

Laravel 安全最佳实践：认证授权、输入验证、CSRF、文件上传、密钥管理、限流和部署加固。

---

## 功能

- 认证和授权（Sanctum、Policies、Gates、策略中间件）
- 密码安全（Hashing、强密码规则）和会话加固
- 输入验证（Form Request）、XSS 防护、SQL 注入防护
- CSRF 保护、文件上传安全、加密字段
- 限流策略（登录、API）、CORS 安全配置
- 安全头（CSP、HSTS）、签名 URL、日志脱敏

## 用法

- `/ecc:laravel-security` - 保护 Laravel 应用免受常见安全漏洞攻击

## 适用场景

- 添加认证和授权机制
- 处理用户输入和文件上传
- 构建新的 API 端点
- 管理密钥和环境配置
- 加固生产环境部署
