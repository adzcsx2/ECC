# /ecc:frontend-patterns

React、Next.js 前端开发模式：组件设计、状态管理、数据请求与性能优化。

---

## 功能

- 组件模式：组合优于继承、复合组件（Compound Components）与 Render Props 模式
- 自定义 Hooks：useToggle、useQuery、useDebounce 等通用 hook 模板
- 状态管理：Context + useReducer 组合模式、不可变更新
- 性能优化：useMemo/useCallback 记忆化、React.memo、代码分割 lazy loading、长列表虚拟化
- 表单处理：受控表单 + 验证 + 错误状态管理模式
- 动效模式：Framer Motion 列表动画、模态框进出动画（AnimatePresence）
- 无障碍：键盘导航组件、焦点管理、语义化 ARIA 属性

## 用法

- `/ecc:frontend-patterns` - 在构建 React 组件、管理状态、优化性能或处理表单时激活

## 适用场景

- 构建 React 组件（组合、props、渲染）
- 管理客户端状态（useState、useReducer、Zustand、Context）
- 实现数据请求（SWR、React Query、Server Components）
- 性能优化（记忆化、虚拟化、代码分割）
- 构建无障碍、响应式 UI 模式
