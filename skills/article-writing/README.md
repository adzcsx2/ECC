# /ecc:article-writing

撰写具有鲜明个人风格的文章、博客、教程和长篇内容。

---

## 功能

- 撰写博客文章、评论、发布公告、指南、教程和 Newsletter
- 根据提供的示例或品牌指南匹配特定人声风格
- 使用证明（示例、数字、截图、代码）替代形容词来建立可信度
- 遵循"先展示再解释"的结构原则
- 禁止通用 AI 过渡语和空洞的套话

## 用法

- `/ecc:article-writing` - 撰写或润色长篇内容，保持一致的风格和观点

## 适用场景

- 撰写博客文章、评论、发布公告、指南或 Newsletter
- 将笔记、转录或研究转化为精炼的文章
- 匹配创始人、运营者或品牌的声音风格
- 收紧已有长篇内容的结构、节奏和论据
