# /ecc:strategic-compact

在逻辑任务边界建议手动压缩上下文，而非依赖自动压缩。

---

## 功能

- 跟踪工具调用次数并在阈值时提醒压缩
- 任务阶段转换指南：何时压缩、何时保留
- 上下文存活清单：CLAUDE.md、TodoWrite、Memory 持久化
- 触发表懒加载模式减少基线上下文 50%+
- Token 优化模式与重复指令检测

## 用法

- `/ecc:strategic-compact` - 策略性上下文压缩建议

## 适用场景

- 长会话接近上下文限制时
- 多阶段任务间的上下文切换
- 调试完成后清理上下文继续新任务
