# /ecc:energy-procurement

电力与天然气采购、电价优化、需求费用管理与可再生能源 PPA 评估的专业知识。

---

## 功能

- 多设施采购策略：固定价格、指数定价、分块指数混合、分层采购的对比与选择
- 需求费用管理：峰值识别、负载转移、电池削峰、需求响应（DR）计划参与
- 可再生能源 PPA 评估：物理 PPA 与虚拟 PPA 的经济模型、基差风险与信用要求分析
- 负载分析：基载 vs 可变载、负载因子、各系统贡献占比的量化方法
- 市场结构分析：受管制 vs 自由化市场、LMP 节点定价、容量市场机制
- 风险管理与预算：分层对冲、天气风险、监管变更风险、供应商信用风险

## 用法

- `/ecc:energy-procurement` - 在进行电力或天然气采购、评估 PPA 方案、优化电价结构或制定能源预算时激活

## 适用场景

- 为多个工厂/设施发起电力或天然气采购 RFP
- 分析电价结构和费率优化机会
- 评估需求费用削减策略（负载转移、电池储能、功率因数校正）
- 审查现场或虚拟可再生能源 PPA 报价
- 制定年度能源预算和对冲策略
- 应对极端天气或监管变更引发的市场价格波动
