# /ecc:liquid-glass-design

iOS 26 Liquid Glass 设计系统实现：动态玻璃材质、模糊反射效果和交互式形变。

---

## 功能

- SwiftUI 玻璃效果 API（glassEffect、glassEffectContainer、glassEffectUnion）
- 形状定制（capsule、rect、circle）和色彩调色（tint）
- 元素间的形变动画和合并效果
- UIKit 支持（UIGlassEffect、UIGlassContainerEffect）
- WidgetKit 集成（accented 渲染模式、widgetAccentable）

## 用法

- `/ecc:liquid-glass-design` - 在 iOS 26+ 应用中使用 Liquid Glass 设计系统构建界面

## 适用场景

- 构建或更新 iOS 26+ 应用的新设计语言
- 实现玻璃风格按钮、卡片、工具栏或容器
- 创建玻璃元素间的形变过渡
- 在 Widget 中应用 Liquid Glass 效果
