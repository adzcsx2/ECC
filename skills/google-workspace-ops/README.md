# /ecc:google-workspace-ops

跨 Drive、Docs、Sheets、Slides 的统一工作流面板，管理计划、追踪器和文档资产。

---

## 功能

- 统一入口：从 Google Drive 搜索定位文件，再切换到专项工具
- 结构化工作流：查找 -> 检查 -> 精准编辑 -> 维护工作系统整洁
- 支持 Docs 精准编辑、Sheets 指定范围和标签操作、Slides 内容与模板迁移
- 输出标准格式：资产信息、当前状态、执行操作、后续任务
- 资产清理：识别重复追踪器、过期文档和应归档/合并的资产

## 用法

- `/ecc:google-workspace-ops` - 管理 Google Workspace 中的共享文档，查找、编辑、清理 Drive 中的文档、表格和幻灯片

## 适用场景

- 在 Google Drive 中查找文档/表格/幻灯片并原地更新
- 整合存储在 Drive 中的计划、追踪器、笔记或客户列表
- 清理或重构共享电子表格
- 导入、修复或重新格式化 Google Slides 演示文稿
- 从 Docs/Sheets/Slides 中提取摘要以辅助决策
