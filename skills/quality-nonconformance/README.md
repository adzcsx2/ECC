# /ecc:quality-nonconformance

受监管制造业（FDA/IATF/AS9100）的质量控制和非一致性管理体系知识，覆盖 NCR 生命周期、根源分析、CAPA 和供应商质量管理。

---

## 功能

- NCR 全生命周期管理：识别、记录、调查、MRB 处置（use-as-is/返工/修理/退货/报废）
- 根源分析方法选择：5 Whys、鱼骨图、故障树分析、8D 方法论
- CAPA 系统：启动、纠正 vs 预防、有效性验证、关闭标准
- SPC 统计过程控制：图表选择、Western Electric 规则、过度调整问题
- 来料检验：AQL 抽样计划、LTPD、跳批资格、CoC 依赖策略
- 供应商质量管理：审核方法、评分卡、纠正措施升级链

## 用法

- `/ecc:quality-nonconformance` - 调查非一致性、执行根源分析、制定 CAPA 计划或解释 SPC 数据时使用

## 适用场景

- 调查来料检验、过程或最终检验中的非一致性
- 使用 5 Whys/鱼骨图/故障树进行根源分析
- 确定非一致材料的处置方式
- 创建或审查 CAPA 计划
- 解读 SPC 数据和控制图信号
- 准备或回应法规审核发现
- 处理供应商质量问题
