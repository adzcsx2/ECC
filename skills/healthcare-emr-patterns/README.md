# /ecc:healthcare-emr-patterns

电子病历系统开发模式：临床安全、诊疗流程、处方生成与无障碍优先 UI。

---

## 功能

- 患者安全优先：药物相互作用必须告警，异常检验值必须高亮，危急生命体征必须触发升级流程
- 单页诊疗流程：垂直滚动布局，无需切换标签页，8 步完整诊疗链路
- 智能模板系统：可点击症状芯片、必填字段、红色警戒非可关闭弹窗
- 用药安全模式：药物检查 -> 相互作用验证 -> 剂量验证 -> 关键交互阻止开方 -> 必须记录覆盖原因
- 签名锁定模式：签后不可编辑，仅能添加补遗，审计追踪完整记录

## 用法

- `/ecc:healthcare-emr-patterns` - 构建电子病历系统，涵盖诊疗流程、临床安全、处方模块、审计追踪和医疗无障碍 UI

## 适用场景

- 构建患者就诊流程
- 实现临床笔记记录
- 设计处方/用药模块
- 集成临床决策支持系统
- 构建实验室结果展示
- 实现临床数据审计追踪
- 设计临床数据输入的无障碍 UI
