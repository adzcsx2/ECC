# /ecc:ck

持久化项目上下文记忆，会话启动自动加载，通过确定性脚本管理项目状态。

---

## 功能

- 项目注册与会话启动自动上下文加载（~100 tokens/会话）
- 会话状态保存：摘要、进行中的工作、下一步、决策记录和阻塞项
- 断点续作：按项目名或序号恢复完整工作上下文
- 免污染设计：用专门脚本操作数据，禁止手动编辑 JSON 或 MD 文件
- 支持 v1 到 v2 数据迁移，保留备份

## 用法

- `/ecc:ck:init` - 注册新项目
- `/ecc:ck:save` - 保存当前会话状态
- `/ecc:ck:resume [name|number]` - 恢复项目上下文
- `/ecc:ck:list` - 查看所有已注册项目
- `/ecc:ck:forget [name|number]` - 删除项目上下文

## 适用场景

- 需要在多个会话间保持项目上下文时
- 跨会话追踪工作进度、决策和阻塞项
- 新会话开始后快速恢复之前的工作状态
