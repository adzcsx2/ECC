# /ecc:security-bounty-hunter

挖掘可提交漏洞赏金的真实安全漏洞，聚焦远程可达的高价值攻击路径。

---

## 功能

- 聚焦远程可达、用户可控的攻击路径（SSRF、认证绕过、RCE、SQL 注入等）
- 自动过滤低信号发现：跳过本地-only 路径、测试代码、硬编码命令等
- 标准化漏洞报告结构：描述、代码位置、PoC、影响、影响版本
- 提交前质量门：确认代码路径可达、输入可控、PoC 有效、无重复

## 用法

- `/ecc:security-bounty-hunter` - 扫描仓库寻找可提交漏洞赏金的安全问题时使用

## 适用场景

- 扫描代码仓库寻找可实际利用的漏洞
- 准备 Huntr、HackerOne 或类似平台的漏洞赏金提交
- 需要判断"这个漏洞真的能拿到赏金吗"而非"理论上安全吗"
