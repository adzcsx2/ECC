# /ecc:remotion-video-creation

Remotion 视频创建最佳实践，包含 29 条领域特定规则，覆盖 3D、动画、音频、字幕、图表、转场等。

---

## 功能

- 3D 内容：Three.js 和 React Three Fiber 集成
- 动画：基础动画技能和文本动画模式
- 多媒体素材：图片、视频、音频、字体导入
- 字幕：SRT 导入、TikTok 风格字幕页、词级高亮
- 数据处理：图表可视化、DOM 节点测量、文本测量和自适应
- 时序控制：延迟、裁剪、持续时间限制、插值曲线和弹性动画
- 转场和序列化模式
- TailwindCSS 在 Remotion 中的应用

## 用法

- `/ecc:remotion-video-creation` - 处理任何 Remotion 代码时自动获取领域知识
- 各子规则文件包含详细说明和代码示例

## 适用场景

- 使用 Remotion 创建视频时
- 需要 3D、动画、字幕或图表功能
- 处理视频素材的导入、裁剪和音频控制
- 设计场景转场和序列化效果
