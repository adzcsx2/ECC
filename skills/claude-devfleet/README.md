# /ecc:claude-devfleet

在隔离工作树中编排多代理并行编码任务，自动合并和报告结果。

---

## 功能

- 项目规划：AI 自动将描述拆分为带依赖链的任务 DAG
- 并行调度：在隔离 git worktree 中最多并发运行 3 个代理
- 自动流转：任务完成后自动合并代码并触发依赖任务
- 结构化报告：每个任务返回文件变更、完成内容、错误和下一步
- 支持全自动、手动分步、顺序审核三种工作模式

## 用法

- `/ecc:claude-devfleet` - 通过 MCP 连接 DevFleet 实例，规划并调度多代理编码项目

## 适用场景

- 需要拆分大型功能并由多个代理并行开发时
- 需要将编码、测试、审查串联为自动化流水线
- 复杂项目需要依赖管理和任务编排时
