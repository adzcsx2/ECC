# /ecc:messages-ops

证据优先的实时消息检索工作流，用于读取短信、私信、验证码及线程审查。

---

## 功能

- 解析消息来源（本地消息、社交私信、浏览器消息等）并定位精确线程
- 先读后写：在生成回复前先检查最新入站消息和未闭环事项
- 专门的一次性验证码检索流程：按时段和发件人快速定位
- 输出标准化的证据报告：来源、发送方、时间窗口和状态

## 用法

- `/ecc:messages-ops` - 读取消息、查找验证码、检视对话线程或证明消息来源

## 适用场景

- 需要从 iMessage、社交媒体私信等实时消息中检索信息
- 查找最近的一次性验证码
- 在回复或跟进前先审阅完整对话上下文
- 注意：如果是邮箱类工作，应使用 `email-ops`
