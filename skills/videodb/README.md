# /ecc:videodb

视频与音频的感知、索引、搜索和编辑——摄入、理解和操作视频内容。

---

## 功能

- 桌面感知：录制屏幕/麦克风/系统音频，实时流式传输上下文和会话记忆
- 视频摄入与流式播放：上传本地文件或 URL，返回可播放的流链接，支持转码归一化
- 多维度索引和搜索：构建视觉/语音/关键词索引，按时间戳返回精确片段并编译为剪辑
- 时间线编辑：字幕生成/翻译/烧录、文字/图片叠加、音频叠加、程序化导出
- 实时 RTSP 流监控：连接实时视频流，运行视觉和语音理解并发出事件告警

## 用法

- `/ecc:videodb` - 摄入视频文件、构建索引、搜索片段、编辑时间线或启动桌面录制

## 适用场景

- 开始/停止桌面会话录制并生成可搜索的时间线和摘要
- 摄入视频并搜索特定场景或对话片段
- 为视频生成字幕、翻译或配音
- 连接 RTSP 摄像头进行实时监控和事件告警
