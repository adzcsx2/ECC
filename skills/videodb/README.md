# /ecc:videodb

VideoDB 视频处理：摄取、索引、搜索、编辑、生成与桌面会话记忆。

---

## 功能

- 视频摄取：本地文件、URL、YouTube、RTSP 直播流
- 桌面会话录制：屏幕+麦克风+系统音频，实时告警与摘要
- 多维度索引：口述词、关键词、场景语义搜索
- 时间线编辑：字幕、叠加层、音乐、配音
- 生成媒体：AI 图片、语音、音乐、音效

## 用法

- `/ecc:videodb` - 视频处理与桌面会话感知

## 适用场景

- 视频内容搜索与片段提取
- 桌面操作录制与事后回放
- 视频字幕生成、翻译与烧录
