# /ecc:pubmed-database

PubMed 和 NCBI E-utilities 生物医学文献检索工作流。

---

## 功能

- PubMed 查询构建：MeSH 词、字段标签、日期与文献类型过滤
- NCBI E-utilities API：esearch、esummary、efetch、elink
- 批量检索的去重与历史服务器参数
- 可复现的检索日志记录

## 用法

- `/ecc:pubmed-database` - 检索 PubMed 生物医学文献

## 适用场景

- 系统综述或文献调研中的 PubMed 检索
- 通过 API 批量获取 PMID 摘要和元数据
- 构建可复现的生物医学文献检索策略
