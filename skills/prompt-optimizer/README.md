# /ecc:prompt-optimizer

分析原始提示词，识别意图和缺失上下文，匹配 ECC 生态组件，输出即用优化版提示词。

---

## 功能

- 6 阶段分析管线：项目检测、意图分类、范围评估、ECC 组件匹配、缺失上下文检测、模型推荐
- 自动检测项目技术栈（Node.js、Go、Python、Rust、Java 等）
- 根据意图+范围+技术栈推荐 ECC 命令、技能和代理
- 输出完整版和快速版两个优化提示词
- 仅提供优化建议，不执行用户的实际任务

## 用法

- `/ecc:prompt-optimizer` - 当说出"优化这个 prompt"、"改进我的提示词"、"怎么写 prompt"时触发
- 中文等价触发词：优化prompt、改进prompt、帮我优化这个指令

## 适用场景

- 用户有一个草稿提示词，想让它更高效
- 不确定如何向 Claude Code 描述复杂任务
- 想知道应该使用哪些 ECC 命令和代理
- 需要为多人协作的大型任务拆分提示词
