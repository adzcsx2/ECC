# /ecc:returns-reverse-logistics

退货与逆向物流全生命周期管理专业知识和决策框架。

---

## 功能

- 退货授权（RMA）与退货政策评估，包含跨渠道、国际退货等复杂场景
- 退货商品检验与分级（A-D 四级），针对不同品类（电子产品、服装、家具等）的差异化标准
- 处置通道经济决策（重新上架、翻新、清仓、报废、退厂），最大化回收价值
- 退货欺诈检测与评分模型，识别调包、串号、序列退货等欺诈模式
- 供应商追偿（RTV）、保修索赔管理与缺陷索赔流程

## 用法

- `/ecc:returns-reverse-logistics` - 处理退货请求、检测欺诈、管理保修索赔或优化逆向物流决策时使用

## 适用场景

- 电商和零售场景下处理产品退货、退款决策、换货流程
- 评估退货欺诈风险并制定反欺诈策略
- 管理供应商缺陷索赔和保修事务
- 优化退货处置通道以平衡客户满意度与利润保护
