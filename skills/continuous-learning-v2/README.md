# /ecc:continuous-learning-v2

基于本能的持续学习系统，通过 Hook 观测会话，用置信度评分自动提取和演化知识。

---

## 功能

- 100% 可靠观测：PreToolUse/PostToolUse hook 而非概率性技能触发，确保每次工具调用都被记录
- 原子本能模型：每个本能包含触发器、动作、置信度（0.3-0.9）、领域标签和证据链
- 项目隔离：v2.1 新增项目作用域，React 模式自动绑定 React 项目，防止跨项目污染
- 演变管线：本能 -> 聚类 -> 演化为完整技能/命令/代理
- 六条管理命令：状态查看、聚类演变、导入导出、跨项目提升、项目列表
- 置信度动态调整：重复观察增加信心，用户纠正降低信心

## 用法

- `/ecc:instinct-status` - 查看当前项目的所有本能及其置信度
- `/ecc:evolve` - 将相关本能聚类并演化为技能或命令
- `/ecc:promote [id]` - 将项目级本能提升为全局本能
- `/ecc:projects` - 列出所有已知项目及其本能数量

## 适用场景

- 希望 Claude Code 从使用中自主学习你的编码偏好时
- 需要在多个项目间共享通用最佳实践但保持语言特定偏好隔离
- 自动将反复出现的工作模式转化为可复用的命令和技能
- 导出高置信度本能供团队共享
