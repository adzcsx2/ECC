# /ecc:springboot-tdd

Spring Boot 测试驱动开发：JUnit 5、Mockito、Testcontainers 与 JaCoCo。

---

## 功能

- TDD RED-GREEN-REFACTOR 完整工作流
- 单元测试：JUnit 5 + Mockito（Arrange-Act-Assert）
- Web 层测试：MockMvc 切片测试
- 集成测试：@SpringBootTest + Testcontainers 真实数据库
- 覆盖率：JaCoCo 配置与 80% 阈值

## 用法

- `/ecc:springboot-tdd` - Spring Boot 测试驱动开发指南

## 适用场景

- 新增功能或 API 端点
- Bug 修复与回归测试
- 重构前确保测试保护
