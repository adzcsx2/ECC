# /ecc:social-graph-ranker

基于加权图的社交网络排名引擎：人脉桥梁评分与熟人推荐路径分析。

---

## 功能

- 加权图排名模型：计算桥梁得分 B(m)，考虑跳数衰减和目标权重
- 二阶扩展分析：挖掘熟人网络中的间接人脉路径
- 响应调整最终排名：结合互动历史对桥梁得分进行加权修正
- 分级输出：Tier 1 直接推荐 -> Tier 2 有条件下推荐 -> Tier 3 直接联系
- 支持 X 和 LinkedIn 双平台图谱数据

## 用法

- `/ecc:social-graph-ranker` - 需要社交网络关系排名引擎本身时使用（非完整外展工作流）

## 适用场景

- 评估"谁是我网络中最好的介绍人"
- 对目标列表绘制熟人推荐路径
- 衡量一度和二度连接的桥梁价值
- 作为 lead-intelligence 和 connections-optimizer 的底层排名引擎
