# /ecc:tinystruct-patterns

tinystruct Java 框架开发模式：@Action 路由、CLI/HTTP 双模、数据持久化。

---

## 功能

- CLI 与 HTTP 同等公民的 @Action 路由标注
- Builder/Builders 原生 JSON 序列化（零外部依赖）
- AbstractData POJO 数据库持久化与 XML 映射
- SSE 服务器推送、文件上传、HTTP 客户端网络调用
- JUnit 5 测试模式与 ActionRegistry 路由调试

## 用法

- `/ecc:tinystruct-patterns` - tinystruct 框架开发模式参考

## 适用场景

- 基于 tinystruct 框架构建 Java 模块
- 调试路由冲突与 CLI 参数解析
- 使用 Builder/Builders 替代 Gson/Jackson
