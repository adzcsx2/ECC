# /ecc:csharp-testing

使用 xUnit、FluentAssertions 和现代测试实践的 .NET 测试模式指南。

---

## 功能

- xUnit + FluentAssertions + NSubstitute/Moq 的完整测试栈
- Arrange-Act-Assert 结构和参数化测试（Theory/MemberData）
- ASP.NET Core 集成测试（WebApplicationFactory）
- Testcontainers 用于真实数据库集成测试
- Test Data Builder 模式减少重复测试数据创建

## 用法

- `/ecc:csharp-testing` - 编写 C# 测试或审查测试质量时获得模式参考

## 适用场景

- 为 C# 代码编写新测试或设置 .NET 测试基础设施
- 审查测试质量和覆盖率
- 调试 flaky 或慢速测试
