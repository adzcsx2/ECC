# /ecc:investor-outreach

投资者沟通文案：冷邮件、暖场介绍、跟进邮件和投资人更新邮件的专业化写作。

---

## 功能

- 五大核心规则：每封邮件个性化、降低请求门槛、用数据代替形容词、保持简洁、不可批量发送
- 冷邮件结构：简短具体的标题 + 投资者专属理由 + 核心卖点 + 一个明确下一步行动
- 个性化来源：相关投资组合公司、公开研究文章/演讲、共同人脉、市场匹配度
- 跟进节奏：初始邮件 -> 第 4-5 天（附新数据点）-> 第 10-12 天（最终跟进干净收尾）
- 硬性禁用词：删掉"I'd love to connect"、"excited to share"等模板化用语

## 用法

- `/ecc:investor-outreach` - 为天使投资人、VC、战略投资者撰写个性化的融资沟通邮件

## 适用场景

- 写投资人冷邮件
- 起草暖场介绍请求
- 发送会议后的跟进或无回复跟进
- 在融资流程中发送投资者更新
- 根据基金方向或合伙人匹配度定制外联
