# /ecc:openclaw-persona-forge

为 OpenClaw AI Agent 锻造完整的龙虾灵魂角色方案。

---

## 功能

- 引导式或抽卡模式创建龙虾角色身份
- 生成身份张力、底线规则、性格缺陷、名字和头像提示词
- 内置 800 万种组合的抽卡引擎 gacha.py
- 支持输出 SOUL.md 和 IDENTITY.md 落地文件
- 可选自动生图（需已安装的生图 skill）

## 用法

- `/ecc:openclaw-persona-forge` - 创建 OpenClaw 龙虾灵魂角色

## 适用场景

- 从零创建 OpenClaw Agent 的角色人格
- 为已有 SOUL.md 补充名字、底线规则和头像
- 抽卡模式快速获得随机龙虾灵魂方案
