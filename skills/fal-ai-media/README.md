# /ecc:fal-ai-media

通过 fal.ai MCP 统一生成图像、视频和音频的 AI 媒体工作流。

---

## 功能

- 图像生成：Nano Banana 2（快速迭代）与 Nano Banana Pro（高保真）的 text-to-image 与图像编辑
- 视频生成：Seedance、Kling Video、Veo 3 等多模型 text-to-video 与 image-to-video 支持
- 音频生成：CSM-1B 自然语音合成、ThinkSound 视频配声、ElevenLabs 专业语音
- 成本预估：`estimate_cost` 可在生成前了解费用
- 模型发现：`search`、`find`、`models` 探索可用模型及其参数
- 完整 MCP 工具链：generate、result、status、cancel、upload

## 用法

- `/ecc:fal-ai-media` - 当需要 AI 生成图像、视频或音频内容时激活

## 适用场景

- 从文字描述生成产品图、概念图或宣传图
- 从文本或图像创建短视频内容
- 生成自然语音旁白或对话语音
- 为视频自动匹配背景音乐或环境音效
- 快速迭代视觉创意时从低成本模型开始，最终切换到高保真模型
