# /ecc:ui-to-vue

将 UI 设计截图批量转换为 Vue 3 组件代码。

---

## 功能

- 批量转换：目录级设计截图 -> Vue 3 Composition API 组件
- 支持 Vant、Element Plus、Ant Design Vue 组件库映射
- 页面分组：列表/详情/表单/加载/空状态合并为一页组件
- 重复 UI 区域自动提取为共享组件
- 切图资源按优先级匹配：页面 > 模块 > 全局

## 用法

- `/ecc:ui-to-vue` - 截图批量转 Vue 3 组件

## 适用场景

- 设计稿到 Vue 3 页面的快速初版生成
- 移动端（Vant）或桌面端（Element Plus/Ant Design Vue）项目
- UI 组件库驱动的批量页面搭建
