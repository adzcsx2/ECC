# /ecc:connections-optimizer

重组社交网络连接，包括关注清理、拓展推荐和个性化外联。

---

## 功能

- 跨平台网络分析：X 关注列表和 LinkedIn 连接的审核与管理
- 三级修剪模式：轻量级（仅高置信度低价值）、默认（平衡修剪）、激进（大范围清理）
- 评分模型：基于互惠性、活跃度、优先级匹配度、网络桥梁价值等正负信号
- 拓展推荐：使用 lead-intelligence 发现高价值关注/连接目标
- 外联草稿生成：在用户真实语调下生成 X DM、LinkedIn 消息和邮件草稿

## 用法

- `/ecc:connections-optimizer` - 按评分模型审核社交网络，生成修剪队列、保留列表和拓展推荐

## 适用场景

- 清理 X 关注的低质量账号
- 让社交网络向当前工作重点靠拢
- 发现新的高价值关注和连接目标
- 基于真实关系而非冷名单发起外联
