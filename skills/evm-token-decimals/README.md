# /ecc:evm-token-decimals

防止 EVM 链上代币小数位不匹配的运行时查询、链感知缓存与安全归一化策略。

---

## 功能

- 运行时查询 ERC-20 代币的 `decimals()` 而非硬编码，避免余额计算偏离数量级
- 按链 ID + 代币地址缓存小数位，而非按代币符号（同一符号在不同链上精度不同）
- 使用 `Decimal`/`BigInt` 精确算术而非浮点数进行价值计算
- 处理非标准代币：`decimals()` 调用失败时的降级策略与日志记录
- Solidity 合约内 18 位 WAD 归一化函数模板
- TypeScript (ethers) 与 Python (web3.py) 的双语言实现参考

## 用法

- `/ecc:evm-token-decimals` - 在读取 ERC-20 余额、计算法币价值、跨链比较代币数量时激活

## 适用场景

- 读取 ERC-20 代币余额并换算为人类可读数量
- 从链上余额计算对应的法币价值
- 跨多条 EVM 链比较同一种代币的数量
- 处理跨链桥接资产（bridge 后精度可能变化）
- 构建投资组合追踪器、交易机器人或聚合器
