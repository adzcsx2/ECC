# /ecc:gget

gget 命令行与 Python 快速基因数据库查询工作流。

---

## 功能

- 快速查询 Ensembl 基因 ID、元数据与序列
- BLAST/BLAT 快速比对，无需本地构建数据库
- 蛋白质结构（AlphaFold/PDB）与通路富集分析
- 多模块集成：search、info、seq、ref、enrichr 等

## 用法

- `/ecc:gget` - 使用 gget 进行基因数据库快速查询

## 适用场景

- 基因信息快速查询与序列获取
- 在转向 Biopython/Snakemake 等重型工具前的初步探索
- 构建可复现的生物信息学证据日志
