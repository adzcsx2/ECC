# /ecc:x-api

X/Twitter API 集成——OAuth 认证、发推、线程、搜索和分析的编程化操作。

---

## 功能

- 两种认证模式：OAuth 2.0 Bearer Token（只读应用）和 OAuth 1.0a（用户上下文读写）
- 核心操作：发推、发线程、读取用户时间线、搜索推文、上传媒体并发布
- 自适应速率限制处理，读取 x-rate-limit-remaining 和 x-rate-limit-reset 头
- 安全规范：密钥环境变量管理、.env 不入仓库、令牌泄露后轮换
- 与内容引擎集成：品牌声线 + 内容生成 + X API 发布的完整内容流水线

## 用法

- `/ecc:x-api` - 通过 X/Twitter API 编程化发布推文、搜索内容和获取分析

## 适用场景

- 程序化发推或发线程
- 读取时间线、提及或用户数据
- 搜索 X 上的内容、趋势或对话
- 构建 X 集成或机器人
- 分析和追踪参与度数据
