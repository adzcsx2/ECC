# /ecc:deep-research

多源深度研究，使用 Firecrawl 和 Exa MCP 工具搜索、阅读、合成并生成带引用的研究报告。

---

## 功能

- 将研究主题拆分为 3-5 个子问题，系统化搜索
- 多源搜索策略：2-3 种关键词变体，混合通用和新闻查询
- 深度阅读关键来源（3-5 篇全文），不依赖搜索摘要
- 结构化输出：执行摘要、主题分节、关键观点、完整来源列表
- 支持并行子代理加速广泛主题研究

## 用法

- `/ecc:deep-research` - 深度研究某个主题，获得带完整引用来源的综合报告

## 适用场景

- 竞争分析、技术评估或市场规模研究
- 公司、投资者或技术的尽职调查
- 用户说"research""deep dive""investigate""current state of"的场景
