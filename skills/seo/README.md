# /ecc:seo

搜索引擎优化全流程：技术 SEO、页面优化、结构化数据、Core Web Vitals、内容策略。

---

## 功能

- 技术 SEO 审计：抓取性、可索引性、canonical 一致性、重定向链检查
- 页面优化：title 标签、meta description、标题层级结构（H1-H3）
- 结构化数据：Article、Product、BreadcrumbList、FAQPage 等 JSON-LD 实现
- Core Web Vitals：LCP < 2.5s、INP < 200ms、CLS < 0.1 的优化策略
- 关键词映射与内链规划：避免内容蚕食，优化锚文本

## 用法

- `/ecc:seo` - 审计网站 SEO、优化页面排名、添加结构化数据时使用

## 适用场景

- 审计网站的抓取性和索引状态
- 优化 title、描述和标题结构以提高搜索排名
- 添加或验证 JSON-LD 结构化数据
- 改善 Core Web Vitals 指标
- 规划关键词映射和内链结构
