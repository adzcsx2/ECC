# /ecc:healthcare-cdss-patterns

临床决策支持系统开发模式：药物相互作用检查、剂量验证、临床评分与警报分类。

---

## 功能

- 药物相互作用检查：双向检查新药与当前用药及过敏史的相互作用，按严重程度排序
- 剂量验证引擎：基于体重、年龄、肾功能的多因素剂量安全验证
- 临床评分：NEWS2、qSOFA 等标准化评分系统，含风险分级和升级指导
- 警报严重程度分类：Critical（阻止操作）、Major（警告弹窗）、Minor（信息提示）
- 患者安全零容忍：假阴性为零容忍，必须 100% 测试通过率

## 用法

- `/ecc:healthcare-cdss-patterns` - 实现临床决策支持功能，包括药物相互作用检查、剂量验证、临床评分和警报系统

## 适用场景

- 实现药物相互作用检查
- 构建剂量验证引擎
- 实现临床评分系统（NEWS2、qSOFA、APACHE、GCS）
- 设计异常临床值的警报系统
- 构建带安全校验的用药医嘱输入
- 将实验室结果解读与临床上下文整合
