# /ecc:uspto-database

USPTO 专利与商标官方数据检索及可复现 IP 研究日志。

---

## 功能

- PatentSearch API：专利与预授权出版物搜索
- TSDR API：商标状态、文档与所有权查询
- 专利文件包（File Wrapper）与审查历史检索
- 专利/商标转让记录查询
- 可复现输出日志表

## 用法

- `/ecc:uspto-database` - 查询 USPTO 专利与商标官方记录

## 适用场景

- 现有技术（prior art）检索与专利组合分析
- 商标状态核查与文档获取
- 构建可复现的 IP 研究记录
