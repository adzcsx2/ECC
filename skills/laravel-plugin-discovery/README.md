# /ecc:laravel-plugin-discovery

通过 LaraPlugins.io MCP 发现和评估 Laravel 包，检查包健康度和版本兼容性。

---

## 功能

- 按关键词、健康度、Laravel/PHP 版本、vendor 搜索 Laravel 包
- 获取包的详细指标（健康评分、最后更新、版本支持、风险评分）
- 检查包的 Laravel 和 PHP 版本兼容性
- 按 vendor 筛选并评估包维护者信誉
- 无需 API key，免费使用

## 用法

- `/ecc:laravel-plugin-discovery` - 搜索和评估 Laravel 扩展包的健康度和兼容性

## 适用场景

- 寻找特定功能的 Laravel 包（如权限、管理面板、API）
- 检查包是否仍在活跃维护
- 验证包的 Laravel 版本兼容性
- 在添加到项目前评估包的质量
