# /ecc:data-scraper-agent

构建全自动 AI 数据采集代理，支持定时抓取、LLM 增强和多种存储后端。

---

## 功能

- 多数据源支持：REST API、HTML 抓取、RSS Feed、JS 渲染页面（Playwright）
- 免费 AI 增强：Gemini Flash 批量分析，多层模型回退链防止配额耗尽
- 反馈学习系统：从用户的正负反馈中学习，持续优化评分精准度
- 多种存储后端：Notion、Google Sheets、Supabase，100% 免费 GitHub Actions 调度运行
- 即用模板：config.yaml 配置驱动，无需修改代码即可定制

## 用法

- `/ecc:data-scraper-agent` - 构建自动监控、采集和分析公开数据的 AI 代理

## 适用场景

- 监控招聘网站、产品价格、新闻或 GitHub 仓库
- 需要定期从多个来源采集数据并用 AI 评分分类
- 想要零托管成本运行的数据采集自动化方案
