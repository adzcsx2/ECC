# /ecc:cost-aware-llm-pipeline

优化 LLM API 调用的成本，通过模型路由、预算跟踪和提示缓存降低开销。

---

## 功能

- 按任务复杂度自动路由到不同成本层级的模型（Haiku 用于简单任务，Sonnet 用于复杂任务）
- 不可变成本追踪器，每次 API 调用返回新状态绝不修改已有数据
- 窄重试逻辑，仅对网络超时、限流和服务器错误重试，认证错误快速失败
- 提示缓存（Prompt Caching）减少重复发送系统提示词的 token 消耗

## 用法

- `/ecc:cost-aware-llm-pipeline` - 在构建调用 LLM API 的应用时，获得成本优化模式指导

## 适用场景

- 构建批量处理大量项目的 LLM 调用流水线
- 需要在预算内完成 API 调用的生产系统
- 多模型架构需要智能路由以平衡成本和质量
