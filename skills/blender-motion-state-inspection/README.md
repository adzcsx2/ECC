# /ecc:blender-motion-state-inspection

检查 Blender 角色、骨骼、姿态、重定向和动画状态。

---

## 功能

- 提取结构化 Blender 状态（骨骼、边界框、接触点、朝向向量），不止依赖截图
- 识别场景中的角色网格、骨骼、辅助物体、摄像机
- 判定前向/上向/侧向轴 —— 使用骨盆、脊椎、肩、头、足综合判断
- 采样关键帧检测接触、着地、空中、极端姿态等状态
- 诊断常见问题：地面穿透、脚步滑动、腿交叉、扭曲损坏、缩放漂移
- 产出区分确认事实、可能原因和修复建议的结构化报告

## 用法

- `/ecc:blender-motion-state-inspection` - 检查 Blender 动画资产的运动状态和模型完整性

## 适用场景

- Blender 角色在动画中看起来扭曲、镜像、压扁、偏移或脚步滑动
- 判断导入的 avatar、骨骼或重定向运动是否匹配预期姿态
- 需要将渲染证据与骨骼、边界框、接触点等结构化事实进行对比
- 判断模型是角色、道具、代理网格、控制骨骼还是损坏的导入
