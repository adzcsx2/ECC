# /ecc:research-ops

基于公开证据的当前状态研究工作流编排器。

---

## 功能

- 智能分类研究需求（事实查询、对比分析、情报收集、监控候选）
- 编排多个研究 skill（exa-search、deep-research、market-research）协同工作
- 明确区分来源事实、用户证据、推断和建议四类信息边界
- 识别重复研究任务并建议转化为监控工作流，避免重复手动搜索
- 优先选择轻量搜索路径，按需升级到深度研究

## 用法

- `/ecc:research-ops` - 当用户需要研究最新信息、对比选项、评估方案时自动激活

## 适用场景

- 需要基于当前公开信息获取事实、对比或推荐时
- 用户已提供部分证据，需要整合后给出新建议时
- 评估某个研究任务是否应转化为定期监控而非一次性查询时
