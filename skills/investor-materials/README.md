# /ecc:investor-materials

创投文档制作：Pitch Deck、一页纸、投资者备忘录、财务模型和融资材料。

---

## 功能

- 黄金法则：所有投资者材料的数据必须内部保持一致，先确认单一事实源再起草
- Pitch Deck 推荐流程：12 页标准结构（公司定位 -> 问题 -> 方案 -> 产品 -> 市场 -> 商业模式 -> 增长 -> 团队 -> 竞争 -> 融资请求 -> 资金用途 -> 附录）
- 一页纸/备忘录：一句话价值主张、时机论证、增长证据前置、精准融资请求
- 财务模型：显式假设、分层收入逻辑、里程碑关联支出、敏感性分析
- 质量门禁：所有数字匹配事实源、资金用途和收入分层求和正确、假设可见不隐藏

## 用法

- `/ecc:investor-materials` - 创建和更新面向投资者的融资材料，确保多份文档间数据一致

## 适用场景

- 创建或修改 Pitch Deck
- 撰写投资者备忘录或一页纸
- 构建财务模型、里程碑计划或资金用途表
- 回答加速器或孵化器申请问题
- 对齐多份融资文档的数据一致性
