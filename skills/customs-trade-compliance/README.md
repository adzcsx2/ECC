# /ecc:customs-trade-compliance

15年以上经验的海关贸易合规专家知识库，涵盖归类、关税优化和制裁筛查。

---

## 功能

- HS 编码归类：按 GRI 规则（1-6）和章节注释进行准确的关税归类
- 关税优化：FTA 利用率、外贸区（FTZ）、临时进口保函、关税退税
- 受限方筛查：SDN、实体清单、欧盟制裁名单的匹配处理和升级协议
- 海关估价：六种估价方法的层次化应用（WTO 估价协议）
- 多辖区覆盖：US CBP、EU 关税同盟、UK post-Brexit 和中国的特殊要求

## 用法

- `/ecc:customs-trade-compliance` - 处理海关归类、关税优化或贸易合规问题时获得专家指导

## 适用场景

- 对进出口商品进行 HS/HTS 编码归类
- 准备海关文件（商业发票、原产地证书、ISF 申报）
- 评估 FTA 资格和关税节省机会
- 回应海关稽查或处罚通知
