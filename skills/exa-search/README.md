# /ecc:exa-search

通过 Exa MCP 进行神经搜索，支持网页、代码示例、公司信息和人物查找。

---

## 功能

- `web_search_exa`：通用网页搜索，支持按类别过滤（公司、研究论文等）和实时爬取
- `get_code_context_exa`：从 GitHub、Stack Overflow 和文档站点获取代码示例和 API 参考
- 公司研究与人物搜索：按 `category:company` 搜索企业信息，按 `site:linkedin.com/in` 找到专业人士
- 灵活的搜索参数：可控制结果数量、内容 token 数、实时爬取策略

## 用法

- `/ecc:exa-search` - 当需要当前网页信息、代码示例、公司调研或个人资料查找时激活

## 适用场景

- 搜索当前新闻、技术趋势或最新信息
- 查找代码示例、API 文档或技术参考
- 调研公司、竞品或市场参与者
- 在特定领域找到专业人士或专家
- 为开发任务进行背景研究
