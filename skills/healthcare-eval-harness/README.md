# /ecc:healthcare-eval-harness

医疗应用部署的安全评估验证套件，单次 CRITICAL 失败即阻止部署。

---

## 功能

- 五类测试分类：CDSS 准确性、PHI 暴露、数据完整性（CRITICAL，100% 通过率）、临床工作流、集成合规（HIGH，95%+）
- CDSS 准确性：验证药物相互作用对（双向）、剂量验证规则、临床评分与公开规范一致性
- PHI 暴露检测：API 错误响应、控制台输出、URL 参数、浏览器存储、跨设施隔离等多维检查
- 数据完整性：签名锁定记录、审计追踪条目、级联删除保护、并发编辑处理
- CI/CD 集成：GitHub Actions 安全检查门禁，CRITICAL 门禁使用 `--bail` 首个失败即停止

## 用法

- `/ecc:healthcare-eval-harness` - 在医疗应用部署前运行安全评估套件，阻止不安全的部署

## 适用场景

- EMR/EHR 应用每次部署前
- 修改 CDSS 逻辑后（药物相互作用、剂量验证、评分）
- 修改涉及患者数据的数据库模式后
- 修改认证或访问控制后
- 配置医疗应用 CI/CD 流水线时
