# /ecc:scholar-evaluation

学术作品结构化评估：论文、提案、综述和引文质量评分。

---

## 功能

- 九维度评分体系：问题、文献、方法、数据、分析、结果、局限性、写作、引文
- 1-5 分量化评估含具体修改建议
- 三种评估范围：全面、定向、比较
- 分离关键阻碍项与修改建议
- 引文交叉验证

## 用法

- `/ecc:scholar-evaluation` - 对学术作品进行结构化质量评估

## 适用场景

- 审查研究论文、提案或学位论文
- 检查论点的证据支撑程度
- 多篇论文的质量或相关性比较
