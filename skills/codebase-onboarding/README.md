# /ecc:codebase-onboarding

系统分析陌生代码库，生成结构化入门指南和项目专属 CLAUDE.md。

---

## 功能

- 四阶段流程：侦查、架构映射、惯例检测、生成产出物
- 侦查阶段并行检测：包清单、框架指纹、入口点、目录结构、配置工具链、测试结构
- 架构映射：技术栈识别、架构模式判断、关键目录映射、请求生命周期追踪
- 惯例检测：命名规范、代码模式、异步风格、Git 提交惯例
- 双重产出：入门指南 + 项目专属 CLAUDE.md（如已存在则增强而非覆盖）

## 用法

- `/ecc:codebase-onboarding` - 分析当前项目代码库，生成入门指南和 CLAUDE.md

## 适用场景

- 首次在项目中使用 Claude Code 时
- 加入新团队或接手不熟悉的代码仓库
- 为新项目自动生成 CLAUDE.md 项目指令
- 用户提出"帮我理解这个代码库"时
