# /ecc:knowledge-ops

跨多存储层（本地文件、MCP memory、向量库、Git 仓库）的知识库管理、摄入和检索。

---

## 功能

- 六层知识架构（活跃执行层 -> Claude Memory -> MCP 知识图谱 -> KB 仓库 -> 外部数据库 -> 本地归档）
- 知识的分类、去重、存储、索引完整流程
- 多源同步（对话记录、GitHub、Linear、浏览器书签等）
- 语义搜索和关系图谱（通过 MCP memory 工具）
- 敏感信息自动脱敏、知识去重保护

## 用法

- `/ecc:knowledge-ops` - 管理、存储、同步和检索跨系统的知识

## 适用场景

- 需要保存信息到知识库中
- 摄入文档、对话或数据到结构化存储
- 在多个知识系统间同步（本地、MCP、Supabase、Git）
- 去重或整理已有知识
