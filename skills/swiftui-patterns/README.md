# /ecc:swiftui-patterns

SwiftUI 架构、状态管理、导航与性能优化最佳实践。

---

## 功能

- 状态管理：@Observable、@State、@Binding、@Environment 选择指南
- 类型安全导航：NavigationStack + NavigationPath
- 视图组合与细粒度重渲染控制
- 性能优化：Lazy 容器、稳定 ID、Equatable 一致性
- #Preview 宏与 Mock 数据快速迭代

## 用法

- `/ecc:swiftui-patterns` - SwiftUI 架构与性能模式参考

## 适用场景

- 构建 SwiftUI 声明式用户界面
- 从 ObservableObject 迁移到 @Observable
- 优化列表和复杂布局的渲染性能
