# /ecc:team-builder

交互式 Agent 选择器，用于组合和并行派发 Agent 团队。

---

## 功能

- 自动发现所有可用 Agent（支持子目录和扁平布局）
- 交互式领域菜单浏览与选择
- 灵活输入：数字、名称、模糊匹配
- 并行派发最多 5 个 Agent 同时执行
- 结果综合：协议、冲突、下一步建议

## 用法

- `/ecc:team-builder` - 组合 Agent 团队并行处理任务

## 适用场景

- 需要多个专业视角审查同一项目
- 组合不同领域 Agent（安全 + SEO + 架构）
- 浏览可用 Agent 后再决定使用哪个
