# /ecc:accessibility

基于 WCAG 2.2 Level AA 标准的无障碍设计与审计工具。

---

## 功能

- 为 Web、iOS、Android 平台生成语义化 ARIA 和无障碍属性
- 审计现有代码的无障碍合规性，识别 POUR（可感知、可操作、可理解、健壮）缺口
- 实现 WCAG 2.2 新标准，如目标尺寸（Minimum）和焦点外观要求
- 提供跨平台无障碍属性映射（ARIA / SwiftUI Accessibility / Compose Semantics）
- 确保键盘导航、屏幕阅读器兼容性和焦点管理

## 用法

- `/ecc:accessibility` - 对 UI 组件进行无障碍审计或生成符合 WCAG 标准的无障碍标注

## 适用场景

- 定义 Web、iOS 或 Android 的 UI 组件规范
- 审计现有代码的无障碍障碍或合规缺口
- 实现 WCAG 2.2 标准如 Target Size 和 Focus Appearance
- 将高层设计需求映射为技术无障碍属性
