# /ecc:nutrient-document-processing

使用 Nutrient DWS API 处理文档的转换、OCR、提取与签名。

---

## 功能

- 文档格式转换（PDF、DOCX、XLSX、PPTX、HTML、图片互转）
- 从 PDF 提取文本和表格数据
- OCR 扫描件识别（支持 100+ 种语言）
- 敏感信息脱敏（SSN、邮箱、信用卡号等）
- 添加水印、数字签名、填充 PDF 表单

## 用法

- `/ecc:nutrient-document-processing` - 通过 Nutrient API 进行文档处理操作

## 适用场景

- 批量文档格式互转
- 扫描件 OCR 文字识别
- 分享前脱敏含 PII 的文档
- 合同数字化签名与表单自动填充
