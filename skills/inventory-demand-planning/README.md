# /ecc:inventory-demand-planning

零售库存需求规划专业经验：预测方法选择、安全库存优化、补货计划和促销效果估算。

---

## 功能

- 多种预测方法选型（移动平均、指数平滑、季节性分解、因果回归、机器学习）及适用场景
- ABC/XYZ 分类矩阵：按价值与可预测性交叉分类，制定差异化补货和安全库存策略
- 安全库存计算：标准公式、服务水平选择、提前期不确定性、新产品预估、间歇性需求处理
- 促销规划：需求信号扭曲处理、Lift 估算、蚕食效应建模、囤货预估和促销后需求低谷
- 决策框架：预测方法选择表、服务水平目标矩阵、降价时机决策、滞销品淘汰触发条件

## 用法

- `/ecc:inventory-demand-planning` - 需求预测、安全库存设置、补货计划和促销效果建模

## 适用场景

- 生成或审查现有/新 SKU 的需求预测
- 基于需求波动和服务水平目标设置安全库存
- 规划季节性切换、促销或新品上市的补货
- 评估预测准确性并调整模型
- 在供应商最小订单量约束或提前期变更下做出采购决策
