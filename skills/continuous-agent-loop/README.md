# /ecc:continuous-agent-loop

自主代理循环模式，支持带质量门槛、评估和恢复控制的持续自动化工作流。

---

## 功能

- 四种循环模式：sequential（默认顺序）、continuous-pr（CI/PR 控制）、rfc-dag（RFC 分解）、infinite（探索并行生成）
- 推荐生产套件：RFC 分解 + 质量门槛 + 评估循环 + 会话持久化
- 故障模式识别：无进展空转、同根因重试、合并队列阻塞、成本漂移
- 恢复机制：冻结循环 -> 审计 -> 缩减范围 -> 提供明确验收标准后重放

## 用法

- `/ecc:continuous-agent-loop` - 选择循环模式，配置自主代理的持续执行流水线

## 适用场景

- 需要代理持续迭代改进代码而不需要人工干预每个步骤时
- 设置带自动化质量检查的 CI/PR 流水线
- 分解大型 RFC 为可并行执行的任务
- 诊断和恢复陷入空转的自主循环
