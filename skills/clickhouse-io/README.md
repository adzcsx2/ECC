# /ecc:clickhouse-io

ClickHouse 数据库的高性能分析查询、表设计和数据工程最佳实践。

---

## 功能

- 存储引擎选型：MergeTree、ReplacingMergeTree、AggregatingMergeTree 的适用场景与建表模式
- 查询优化：分区裁剪、物化视图、聚合函数、窗口函数的高效使用
- 数据插入模式：批量插入与流式插入的最佳实践
- 数据流水线：ETL 模式、CDC 变更数据捕获的实现参考
- 性能监控：慢查询分析、表统计、合并操作检查

## 用法

- `/ecc:clickhouse-io` - 获取 ClickHouse 的查询优化、表设计和数据工程指导

## 适用场景

- 设计 ClickHouse 表结构时需要指导
- 编写分析查询需要优化建议
- 大量数据入库需要最佳实践
- 从 PostgreSQL/MySQL 迁移分析负载到 ClickHouse
