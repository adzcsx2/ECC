# /ecc:bun-runtime

Bun 作为运行时、包管理器、打包器和测试运行器的完整使用指南。

---

## 功能

- Bun 运行时：Node.js 兼容的快速运行时，原生支持 TypeScript
- 包管理：`bun install` 显著快于 npm/yarn，支持 `bun.lock` 锁文件
- 内置打包器与转译器，无需额外配置
- 内置测试运行器 `bun test`，兼容 Jest API
- 从 Node 迁移指南与 Vercel 部署配置

## 用法

- `/ecc:bun-runtime` - 获取 Bun 的使用模式、最佳实践和迁移建议

## 适用场景

- 新 JS/TS 项目评估 Bun 作为运行时
- 从 Node.js 迁移到 Bun 时
- 配置 Vercel 上的 Bun 运行时
- 使用 Bun 内置测试运行器编写和运行测试
