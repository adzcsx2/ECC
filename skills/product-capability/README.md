# /ecc:product-capability

将产品需求转化为可执行的工程能力计划，输出显式的约束、接口、不变量和未决问题。

---

## 功能

- 将 PRD、路线图或产品讨论转化为 SRS 级别的能力合约
- 提取业务规则、作用域边界、不变量和信任边界
- 定义实现面合约：参与者、接口、状态转换、数据模型影响
- 标记非目标和未解决的问题
- 输出可跨会话重用的持久化能力文档

## 用法

- `/ecc:product-capability` - 当产品意图明确但实现约束仍不清晰时使用
- 读取 PRD 或讨论内容，自动生成能力计划文档

## 适用场景

- 功能跨多个服务、仓库或团队，需要能力合约
- 产品意图清晰但架构、数据、生命周期含义模糊
- 高级工程师在审查中反复重申相同的隐含假设
- 需要可在不同平台和会话间复用的能力文档
